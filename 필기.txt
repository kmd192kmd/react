React
- 사용자 화면(UI)를 만들기 위한 JavaScript 라이브러리이다.
- 컴포넌트 지원
    - 복잡한 화면을 작은 부품으로 나누고, 
      데이터가 바뀌면 화면을 자동으로 다시 그린다.
- 메타(페이스북) > 2013년





React 개발 환경
- 직접 개발 > 관련된 라이브러리(*.js) 참조 + 기본 골격 코드 작성
- Node.js(NPM == Maven, Gradle > 라이브리 관리)
- 빌드 도구
    - create-react-app: 구형
    - vite: 현재(프로젝트 관리 도구 - 생성/관리/테스트/빌드/실행)

1. node.js 설치
    - https://nodejs.org

2. vite 설치
    - $ npm install -g create-vite //요즘엔 잘 설치 안함
    - Vite(프랑스어, 비트): 빠른



Hello World

    - npm <옵션>

    - --

    - --template react
        - vite에게 전달하는 옵션


프로젝트 접속하기

프로젝트 중단하기

- $ cd hello
- $ npm run dev //vite 개발 서버 실행 + 프로젝트 동작


Vite + React 프로젝트 표준 구조
- node_modules > React 관련/ Vite 관련 설치된 패키지(npm이). 
               > 절대 수정 금지
- public > 정적 파일(favicon, 로고 이미지 등) 관리
         > 주로 파일명을 그대로 유지해야 하거나, import하지 않은 파일들
- src > 모든 개발 관련 소스들
      - assets > 이미지, 아이콘 등 리소스 관리 
               > Vite가 번들링하는 대상
               > 최적화 대상
               > 파일명 해시 추가됨 

      - main.jsx //시작점 역할. 렌더링 진입점. React 앱을 시작
      - index.css //전역 CSS

      - App.jsx //샘플 페이지
      - App.css

- index.html > React 앱의 진입점
- package.json > 프로젝트 메타 데이터, 스크립트, 의존성 등
               > pom.xml(Maven), build.gradle(Gradle)
- package-lock.json > package.json > 실제 설치된 명세서
- vite.config.js > Vite 빌드 설정
